Flowbite

Open-source UI components built with Tailwind CSS. 56+ components with dark mode support.

Installation

npm install flowbite

Add to your CSS:

@import "flowbite";

Add to your tailwind.config.js:

module.exports = {
  content: [
    "./node_modules/flowbite/**/*.js"
  ],
  plugins: [
    require('flowbite/plugin')
  ],
}

Include the JS (or import in your bundle):

<script src="../path/to/flowbite/dist/flowbite.min.js"></script>
// or
import 'flowbite';

Usage Patterns

Flowbite components can be used three ways:

  1. Data attributes — preferred, zero JS required
  2. Init functions — programmatic initialization
  3. ESM/CJS imports — full control with TypeScript support

Modal JavaScript API

import { Modal } from 'flowbite'

const $modalEl = document.querySelector('#myModal')
const modalOptions = {
  placement: 'bottom-right',
  backdrop: 'dynamic',
  backdropClasses: 'bg-gray-900/50 dark:bg-gray-900/80 fixed inset-0 z-40',
  closable: true,
  onHide: () => { console.log('modal is hidden') },
  onShow: () => { console.log('modal is shown') },
  onToggle: () => { console.log('modal has been toggled') },
}

const modal = new Modal($modalEl, modalOptions)
modal.show()
modal.hide()
modal.toggle()
modal.isHidden()  // returns boolean
modal.isVisible() // returns boolean

drawer.toggle()
